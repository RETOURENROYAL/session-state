⚙ n8n Setup — R3 GitHub Workflows

Startet n8n lokal und importiert alle Workflows:

# 1. n8n starten (Docker)
docker run -d --name n8n \
  -p 5678:5678 \
  -e N8N_BASIC_AUTH_ACTIVE=false \
  -e WEBHOOK_URL=http://localhost:5678 \
  -e GENERIC_TIMEZONE=Europe/Berlin \
  -e N8N_DEFAULT_LOCALE=de \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n

# 2. Warten bis bereit, dann Workflows importieren
sleep 5 && open http://localhost:5678
